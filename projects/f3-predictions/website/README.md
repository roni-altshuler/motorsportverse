# RaceIQ F3 website

The Formula 2 prediction site — a Next.js 16 static export reusing the
MotorsportVerse design system, re-themed to the F3 azure accent (`#1E9BD7`).

## Pages

| Route | Content |
|---|---|
| `/` | Hero, title-fight favourite + top-5 title odds, next-round podium |
| `/standings` | Driver + team championship tables (client-tabbed) |
| `/calendar` | Full season calendar with completed/upcoming status |
| `/predictions` | Next round: predicted race result with win/podium bars + projected qualifying |

## Data

All pages read `public/data/f3.json`, generated by the Python pipeline:

```bash
# from repo root, after installing the F3 package + core/data:
python -m f3_predictions.export        # writes website/public/data/f3.json
```

## Develop

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # static export -> out/
```

## Reuse

~77% of the website is the shared MotorsportVerse design system (`components/ui`,
`components/magicui`, `lib/motion`, `tokens.css`) copied verbatim; only the pages,
nav, and data layer (~23%) are F3-specific. See
[docs/F3_READINESS.md](../../../docs/F3_READINESS.md).
