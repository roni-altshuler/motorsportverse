# Indycar Predictions

Race forecasts for the NTT IndyCar Series (road, street, oval).

Generated from the MotorsportVerse project template. Implement `datasource.py`
and `predict.py`, then reuse the shared core (calibration, registry, drift,
promotion, eval) unchanged. See [docs/adding-a-sport.md](../../docs/adding-a-sport.md).
